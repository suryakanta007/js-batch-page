4\. Course Projects (Portfolio-Worthy)

Students will build 4 professional-grade projects that showcase
real-world JavaScript skills:

Project 1: E-Commerce Product Card with Interactive Features

**What You\'ll Build:**

A responsive product card with add-to-cart functionality, image
carousel, size/color selectors, and dynamic pricing.

**Skills Learned:**

-   DOM manipulation and event handling

-   Working with arrays and objects

-   State management in vanilla JavaScript

Project 2: AI Chatbot Clone (Conversational Interface)

**What You\'ll Build:**

A ChatGPT-style interface with message threading, typing indicators, and
local storage persistence.

**Skills Learned:**

-   Asynchronous JavaScript (promises, async/await)

-   API integration and fetch requests

-   LocalStorage and data persistence

Project 3: Photo Editing Website (Filters & Effects)

**What You\'ll Build:**

An image editor with brightness, contrast, saturation controls, crop
functionality, and download feature.

**Skills Learned:**

-   Canvas API manipulation

-   File handling and image processing

-   ES6+ features (destructuring, arrow functions, template literals)

Project 4: Kanban Board (Drag-and-Drop Task Manager)

**What You\'ll Build:**

A Trello-style board with drag-and-drop tasks, add/edit/delete
functionality, and progress tracking.

**Skills Learned:**

-   Drag-and-drop API

-   Object-oriented JavaScript (classes, constructors)

-   Advanced DOM manipulation and event delegation
